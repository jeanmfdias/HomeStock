# HomeStock

## Version 0.0.1

This project should be a home stock to control, organize and make shopping list from market items, medicine, cleaning items, personal care items.

# Rules to products

- Control expiration date
- Control by unit

# Details to products

- Name
- Brand (Optional)
- Category (Market, Car, Vegetable & Fruits, Meat, Cleaning, Hygiene, Medicine)
- Shop place
- Expiration date (Required)
- Quantity (Unit)
- Emergency Quantity (Quantity that need to achieve to in shopping list)

# Screens

- CRUD to Product
- Report of Products with expiration date approaching
- Report of Products to Buy (Shopping List)

# Authentication & Authorization

The login should be by email, don't necessary (oauth, magic link, password).
Each email should be an user.
Because the idea is to use in a local server.

# Tech Skills

- Docker to Infrastructure (docker-compose to run local and docker-compose to run in production with the optimizations)
- PHP 8.5 with Symfony 8.0
- Vue.js to Frontend
- Unit test to all feature
- Sqlite to Database
- Git CI
